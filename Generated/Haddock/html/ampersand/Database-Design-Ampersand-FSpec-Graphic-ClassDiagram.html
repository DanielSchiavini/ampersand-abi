clAnalysis :: FSpec -> ClassDiag

This function makes the classification diagram. It focuses on generalizations and specializations.

cdAnalysis :: FSpec -> ClassDiag

This function, cdAnalysis, generates a conceptual data model. It creates a class diagram in which generalizations and specializations remain distinct entity types. This yields more classes than plugs2classdiagram does, as plugs contain their specialized concepts. Properties and identities are not shown.

tdAnalysis :: FSpec -> ClassDiag

This function generates a technical data model. It is based on the plugs that are calculated.

classdiagram2dot :: Options -> ClassDiag -> DotGraph String

translate a ClassDiagram to a DotGraph, so it can be used to show it as a picture.
