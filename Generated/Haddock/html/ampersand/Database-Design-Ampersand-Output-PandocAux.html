latexEscShw escapes to LaTeX encoding. It is intended to be used in LaTeX text mode. For more elaborate info on LaTeX encoding, consult the The Comprehensive LATEX Symbol List on: http:/ftp.snt.utwente.nlpubsoftwaretexinfosymbolscomprehensivesymbols-a4.pdf
